# Peti.PRO — Petições em segundos

Um produto da Conecte.se.

## Deploy na Vercel

1. Importe este repositório na Vercel
2. Em Settings → Environment Variables, adicione:
   - `ANTHROPIC_API_KEY` = sua chave da Anthropic Console
3. Clique em Deploy

## Testar localmente

Abra o `index.html` diretamente no navegador para testar a interface.
A geração de petições só funciona com a API key configurada na Vercel.

## Suporte

Fale com a Conecte.se via WhatsApp.
